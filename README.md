## ファイル構成

### HTML

`index.html`

トップページとして表示されるサイト。

`map.html`

index.htmlに埋め込みとして利用されている。主にマップ画像。

##

### CSS 

`index.css`

UIなどの装飾。

`map.css`

マップ画像のサイズ調整などに使用。

### JavaScript

`index.js`

UIの動作などをメインに処理。

`map.js`

埋め込みのズームや教室強調表示が組み込まれている。

`data.js`

変数専用。画像内の教室の座標が原始的に組み込まれている。
基本的には教室番号を割り当て、各階の80番台には主要な階段を割り当てている。
